title: "Credits & pricing"
description: "How prepaid billing credits work."
---

Each enrichment credit costs **3¢**. Enrichment APIs and Playbooks Enrichments share the same prepaid Billing balance.

| API | Credits |
| --- | --- |
| Most company lookups | 1 per call |
| Lookalikes | 1 × companies returned (max 500) |
| Find emails | 1 per domain with emails found |
| Find employee | 0 / 1 / 2 (none / found / +email) |

Check balance via [Get credits](/guides/get-credits) or [Billing status](/agents/billing-status).
