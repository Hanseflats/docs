Remote MCP for Lens. Same API key as the REST API — no admin access, no internal portal tools.

## Endpoint

```text
https://mcp.lens.ly/mcp
```

## Auth

| Method | Example |
| --- | --- |
| Bearer | `Authorization: Bearer sk_live_…` |
| Header | `x-api-key: sk_live_…` |
| Query (`mcp-remote`) | `?api-token=sk_live_…` |

Get a key via [Authentication](/authentication), the dashboard, or [Agent self-serve](/agents/overview).

## Cursor / Claude Desktop

```json
{
  "mcpServers": {
    "lens": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://mcp.lens.ly/mcp?api-token=${env:LENS_API_KEY}"
      ]
    }
  }
}
```

Set `LENS_API_KEY` to your `sk_live_…` key in the environment.

## What you can do

| Area | Tools |
| --- | --- |
| Meta | `lens_overview`, `tools_catalog` |
| Billing | `billing_get`, `billing_setup`, `billing_topup`, `credits_get` |
| Audiences | `audiences_list_plays`, `audiences_get_play`, `audiences_validate`, `audiences_launch`, `audiences_list`, `audiences_get`, `audiences_update`, `audiences_export`, `audiences_sync` |
| Enrichments | `enrichments_list_plays`, `enrichments_get_play`, `enrich_run` (sync), `enrichments_create` / `_launch` / `_get` / `_download` / `_list` (batch) |
| Scrapers | `scrapers_list`, `scrapers_run`, `scrapers_get_job`, `scrapers_get_results` |
| Integrations | `integrations_list`, `integrations_instantly_*`, `integrations_email_finders_get`, `integrations_email_finders_put` |

Audience and enrichment **plays are loaded live** from the API — new dashboard plays show up in MCP automatically.

Scrapers cover TrustRadius, LinkedIn profile/jobs, Indeed, G2, and the rest of the [Scrapers](/api/scrapers) catalog. `scrapers_run` returns sync JSON when possible, or `{ job_id }` for async jobs — then poll with `scrapers_get_job` and page with `scrapers_get_results`.

## Suggested agent flow

<Steps>
  <Step title="billing_get">Check balance and payment method</Step>
  <Step title="billing_setup">If needed — open `setup_url` once in a browser</Step>
  <Step title="Discover">`audiences_list_plays`, `enrichments_list_plays`, or `scrapers_list`</Step>
  <Step title="Validate">`audiences_get_play` / `enrichments_get_play`, or per-scraper docs under `/api/{slug}`</Step>
  <Step title="Run">`audiences_launch`, `enrich_run` / `enrichments_create`, or `scrapers_run`</Step>
  <Step title="Collect">Poll, then export, download, or `scrapers_get_results`</Step>
</Steps>
