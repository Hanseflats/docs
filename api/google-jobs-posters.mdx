Launch via the [Audiences API](/api/audiences) with `play_slug: "google-jobs-posters"`.

Search Google Jobs for companies actively hiring for roles that mention specific tools or use cases. Companies hiring for GTM or outbound roles are high-intent — the posting reveals what they are building and where budget is going. Fresh daily as new jobs appear.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `keyword` | Yes | Google Jobs search query. Use quotes for exact phrases.. Example: `"GTM engineer" "Clay"` |
| `personas` | Yes | Comma-separated roles to find at each company.. Example: `Founder, CEO` |
| `date_window` | No | How recent job postings should be (e.g. last\_30\_days).. Example: `last_30_days` |
| `job_type` | No | all or full\_time. Default: full\_time.. Example: `full_time` |
| `max_leads` | No | Stop after this many companies (prefer launch number\_of\_leads). Example: `30` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `google-jobs-posters` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/google-jobs-posters \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: google-jobs-posters-1" \
  -d '{
  "play_slug": "google-jobs-posters",
  "source_params": {
    "keyword": "\"GTM engineer\" \"Clay\"",
    "personas": "Founder, CEO"
  },
  "number_of_leads": 100,
  "title": "Google Jobs Signal Posters"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Google Jobs Signal Posters",
    "play_slug": "google-jobs-posters",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=leads`.

## Credits

**1 credit per person** resolved (person-first play). Charged as leads are found, not at launch.

## Notes

- Personas always required. Default job\_type=full\_time.
- Prefer top-level number\_of\_leads over max\_leads.
- Billing mode: person (1 credit / person found). Catalog credit\_cost\_per\_unit=1.
- source=google-jobs; volume=10-50; cadence=daily.
