Estimated monthly paid traffic. Returns only paid traffic.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/paid-traffic
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/paid-traffic \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"monday.com"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "monday.com",
    "monthly_paid_traffic": 310000
  },
  "credits_used": 1
}
```

## Credits

1 credit per lookup.

Charged against your prepaid Billing balance at **3¢ per credit**. Same ledger as Playbooks Enrichments.

## Errors

| Status | Meaning |
| --- | --- |
| `400` | Invalid or missing request fields |
| `401` | Invalid API key |
| `402` | Insufficient billing balance — follow `payment.setup_url` if present |
| `500` | Internal server error |
