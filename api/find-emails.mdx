Find email addresses by crawling a company website.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/find-emails
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `url` | Yes | Domain or URL to scrape |
| `limit` | No | Pages to crawl (default 10, max 20) |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/find-emails \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"example.com"}'
```

## Response

```json
{
  "url": "example.com",
  "first_email": "contact@example.com",
  "all_emails": ["contact@example.com", "info@example.com"],
  "pages_crawled": 10,
  "status": "success"
}
```
