Bulk find a current employee per domain by job title. Uses RevenueBase (verified emails only). Pass up to 200 domains per request.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/find-employee
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domains` | Yes | Array of company domains (max 200). Aliases: `urls`, `company_domains`. A single `domain` / `company_website` string is also accepted. |
| `job_title` | Yes | String or array of up to 3 titles (tried as OR match; best title match wins per domain) |

Email is always returned when a match is found (RevenueBase `EMAIL_STATUS_PER = VALID`).

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/find-employee \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "domains": ["hubspot.com", "notion.so"],
    "job_title": ["Head of Growth", "VP Growth"]
  }'
```

1 credit × domains in the request (max 200), including domains with no match.
