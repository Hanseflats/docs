Start a scraping job to resolve reviewer identities from G2, TrustRadius, or Capterra.

## Endpoint

```
POST https://app.lens.ly/api/v1/reviews/resolve
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `url` | Yes | Company domain or review platform URL |
| `reviews` | Yes | Number of reviews to scrape (10–500) |
| `webhook_url` | No | URL to receive results |
| `min_rating / max_rating` | No | Filter by rating |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/reviews/resolve \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"hubspot.com","reviews":100}'
```

## Response

```json
{
  "job_id": "job_abc123",
  "status": "queued",
  "platform": "g2",
  "reviews_requested": 100,
  "max_credits_reserved": 200
}
```

## Credits

| --- | --- |
