Launch via the [Audiences API](/api/audiences) with `play_slug: "new-hires"`.

Find people who recently started a role. Filter by job title, geography, industry, company size, and how many days ago they started. Deliver contacts with work email and LinkedIn via webhook, CSV, or Instantly.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `personas` | Yes | Comma-separated titles to match.. Example: `Chief Marketing Officer, VP Marketing` |
| `geographies` | No | Markets US/UK/EU and/or ISO country codes for company HQ. Default: US.. Example: `US,GB` |
| `companySizes` | No | Employee size ranges.. Example: `11-50,51-200` |
| `industries` | No | LinkedIn industry catalog values (JSON array string).. Example: `["Software Development"]` |
| `min_days_ago` | No | Exclude people who started more recently than this. Default: 30.. Example: `30` |
| `max_days_ago` | No | Only include people who started within this many days. Default: 90.. Example: `90` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `new-hires` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/new-hires \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: new-hires-1" \
  -d '{
  "play_slug": "new-hires",
  "source_params": {
    "personas": "Chief Marketing Officer, VP Marketing"
  },
  "number_of_leads": 100,
  "title": "New Hires"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "New Hires",
    "play_slug": "new-hires",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=leads`.

## Credits

**1 credit per person** resolved (person-first play). Charged as leads are found, not at launch.

## Notes

- Person-first. personas required. Defaults: geographies=US, min\_days\_ago=30, max\_days\_ago=90.
- industries is LinkedIn catalog JSON array string.
- Billing mode: person (1 credit / person found). Catalog credit\_cost\_per\_unit=1.
- source=revenuebase; volume=50-5000; cadence=None.
