Find open job positions at a company from their domain.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/find-jobs
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `company_domain` | Yes | Company domain (e.g. clay.com) |
| `department` | No | Filter by department |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/find-jobs \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"company_domain":"clay.com"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "company": "Clay",
    "domain": "clay.com",
    "is_hiring": true,
    "jobs": [{ "title": "Software Engineer, Backend", "department": "Engineering" }],
    "total_openings": 15
  },
  "credits_used": 1
}
```
