Bulk estimated monthly organic traffic. Pass up to 200 domains per request.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/organic-traffic
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domains` | Yes | Array of company domains (max 200). Aliases: `urls`, `company_domains`. A single `domain` string is also accepted. |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/organic-traffic \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domains":["stripe.com","square.com"]}'
```

## Response

```json
{
  "success": true,
  "results": [
    {
      "domain": "stripe.com",
      "resolved_domain": "stripe.com",
      "monthly_organic_traffic": 1800000
    }
  ],
  "stats": {
    "total_domains": 2,
    "matched": 2,
    "credits_used": 2
  }
}
```

## Credits

1 credit × domains in the request (max 200).

Charged against your prepaid Billing balance at **3¢ per credit**. Same ledger as Playbooks Enrichments.

## Errors

| Status | Meaning |
| --- | --- |
| `400` | Invalid or missing request fields |
| `401` | Invalid API key |
| `402` | Insufficient billing balance - follow `payment.setup_url` if present |
| `500` | Internal server error |
