Estimated monthly organic traffic. Returns only organic traffic.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/organic-traffic
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/organic-traffic \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"stripe.com"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "stripe.com",
    "monthly_organic_traffic": 1800000
  },
  "credits_used": 1
}
```
