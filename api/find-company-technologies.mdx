title: "Find technologies"
description: "Bulk look up company technographics. Up to 200 domains per request."
---

Bulk look up company technographics (CRM, CMS, marketing stack, cloud, etc.) by domain. Pass up to 200 domains per request.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/technologies
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domains` | Yes | Array of company domains (max 200). Aliases: `urls`, `company_domains`. A single `domain` / `company_domain` string is also accepted. |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/technologies \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domains":["hubspot.com","shopify.com"]}'
```

## Response

```json
{
  "success": true,
  "results": [
    {
      "domain": "hubspot.com",
      "found": true,
      "matched_domain": "hubspot.com",
      "company_name": "HubSpot",
      "technologies": {
        "crm": ["HubSpot"],
        "marketing_automation": ["HubSpot"]
      },
      "all": ["HubSpot"]
    }
  ],
  "stats": {
    "total_domains": 2,
    "matched": 1,
    "credits_used": 2
  }
}
```

## Credits

1 credit × domains in the request (max 200). Charged for every domain, even when no tech data is found.

Charged against your prepaid Billing balance at **3¢ per credit**. Same ledger as Playbooks Enrichments.

| --- | --- |

| `401` | Invalid API key |
