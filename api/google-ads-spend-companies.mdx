Launch via the [Audiences API](/api/audiences) with `play_slug: "google-ads-spend-companies"`.

Find US companies by estimated monthly Google ads spend. Not Meta/LinkedIn dollar spend. Filter by minimum spend and company size. Optionally match buyer personas.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `min_value` | Yes | Threshold: 1000, 5000, 10000, 25000, or 50000. Example: `10000` |
| `companySizes` | No | Comma-separated size ids, e.g. 11-50,51-200,201-500. Example: `11-50,51-200,201-500` |
| `include_personas` | No | true to match contacts after companies. Default false.. Example: `false` |
| `personas` | No | Job titles when include\_personas is true. Example: `Founder, Head of Growth` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `google-ads-spend-companies` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/google-ads-spend-companies \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: google-ads-spend-companies-1" \
  -d '{
  "play_slug": "google-ads-spend-companies",
  "source_params": {
    "min_value": "10000"
  },
  "number_of_leads": 100,
  "title": "Companies With Google Ads Spend"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Companies With Google Ads Spend",
    "play_slug": "google-ads-spend-companies",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=companies`.

## Notes

- min\_value thresholds: 1000|5000|10000|25000|50000 (default normalize 10000). Estimated Google \$ spend, not Meta/LinkedIn.
- US filter; personas when include\_personas=true.
- Billing mode: company (1 / company; \+1 per persona match when personas on → typically 2). Catalog credit\_cost\_per\_unit=1.
- volume=100-5000; cadence=monthly.
