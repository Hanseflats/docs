Launch via the [Audiences API](/api/audiences) with `play_slug: "companies-running-google-ads"`.

Find companies that are actively running Google Ads. Filter by size, geography, industry, and funding. Optionally add buyer personas after a company matches. Deliver companies via webhook or CSV.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `companySizes` | No | Employee size ranges, comma-separated.. Example: `51-200,201-500` |
| `geographies` | No | Markets US/UK/EU and/or ISO country codes.. Example: `US,GB` |
| `industries` | No | Industry keywords (OR).. Example: `Software,Marketing` |
| `funded` | No | true / false / omit for either.. Example: `true` |
| `output_mode` | No | companies\_only (default) or companies\_and\_personas.. Example: `companies_only` |
| `personas` | No | Required when output\_mode=companies\_and\_personas.. Example: `Founder, VP Sales` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `companies-running-google-ads` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/companies-running-google-ads \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: companies-running-google-ads-1" \
  -d '{
  "play_slug": "companies-running-google-ads",
  "source_params": {
    "companySizes": "51-200,201-500",
    "geographies": "US,GB"
  },
  "number_of_leads": 100,
  "title": "Companies Running Google Ads"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Companies Running Google Ads",
    "play_slug": "companies-running-google-ads",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=companies`.

## Credits

**1 credit per company** found; **2** if a buyer persona is matched. Charged as results are found, not at launch.

## Notes

- Uses output\_mode (not include\_personas): companies\_only (default) | companies\_and\_personas.
- personas required when output\_mode=companies\_and\_personas.
- Billing mode: company (1 / company; \+1 per persona match when personas on → typically 2). Catalog credit\_cost\_per\_unit=1.
- source=google-ads; volume=50-5000; cadence=None.
