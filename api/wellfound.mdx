title: "Wellfound"

---

Pull founders and team members for a Wellfound company as structured JSON.

## Endpoint

```text
POST https://app.lens.ly/api/v1/scrapers/wellfound
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `slug` | Conditional | Company slug |
| `url` | Conditional | `wellfound.com/company/{slug}` URL |
| `find_linkedin` / `find_email` | No | Enrich people (LinkedIn often already present) |
| `webhook_url` / `delivery` | No | Async delivery options |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/scrapers/wellfound \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"slug":"airbnb"}'
```
