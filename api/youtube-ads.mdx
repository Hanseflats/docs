Check Ads Transparency for YouTube video ads.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/youtube-ads
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/youtube-ads \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"duolingo.com"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "duolingo.com",
    "running_youtube_ads": true,
    "video_count": 4
  },
  "credits_used": 1
}
```
