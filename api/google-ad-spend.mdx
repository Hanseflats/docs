Estimated monthly Google ad spend (RevenueBase). Returns only ad spend — not traffic fields.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/google-ad-spend
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/google-ad-spend \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"shopify.com"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "shopify.com",
    "company_name_rb": "Shopify",
    "monthly_google_adspend": 120000
  },
  "credits_used": 1
}
```

## Credits
