Launch via the [Audiences API](/api/audiences) with `play_slug: "substack-keyword-posts"`.

Enter keywords to find Substack posts on those topics, then collect people who comment. Best for capturing engaged readers around a specific theme.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `keywords` | Yes | Topics used to discover relevant Substack posts.. Example: `sales automation, founder-led sales` |
| `recent_days` | No | Only process posts and comments from the last N days. Default: 30.. Example: `30` |
| `max_leads` | No | Stop after this many commenter leads in one run.. Example: `200` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `substack-keyword-posts` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/substack-keyword-posts \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: substack-keyword-posts-1" \
  -d '{
  "play_slug": "substack-keyword-posts",
  "source_params": {
    "keywords": "sales automation, founder-led sales"
  },
  "number_of_leads": 100,
  "title": "Substack Posts"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Substack Posts",
    "play_slug": "substack-keyword-posts",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=leads`.

## Credits

**1 credit per person** resolved (person-first play). Charged as leads are found, not at launch.

## Notes

- keywords required. Collects commenters on matching posts. Person-first.
- Defaults recent\_days=30; discovery\_mode=posts.
- Billing mode: person (1 credit / person found). Catalog credit\_cost\_per\_unit=1.
- source=substack; volume=20-500; cadence=weekly.
