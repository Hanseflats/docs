Launch via the [Audiences API](/api/audiences) with `play_slug: "jd-replace-companies"`.

Enter a product or vendor name (e.g. Pardot, QuickBooks, Zendesk). We search Greenhouse, Ashby, and Lever job posts for companies that say they are migrating or replacing that product, resolve the hiring company, and optionally match buyer personas. High-intent displacement signal timed to hiring.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `product` | Yes | Named tool or vendor companies are migrating away from. Comma-separate for multiple.. Example: `Pardot` |
| `include_personas` | No | true to match contacts by job title after companies are found. Default: false.. Example: `false` |
| `personas` | No | Job titles when include\_personas is true.. Example: `Controller, VP Finance, Founder` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `jd-replace-companies` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/jd-replace-companies \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: jd-replace-companies-1" \
  -d '{
  "play_slug": "jd-replace-companies",
  "source_params": {
    "product": "Pardot"
  },
  "number_of_leads": 100,
  "title": "Companies Replacing a Product"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Companies Replacing a Product",
    "play_slug": "jd-replace-companies",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=companies`.

## Credits

**1 credit per company** found; **2** if a buyer persona is matched. Charged as results are found, not at launch.

## Notes

- Requires product (named tool being replaced); comma-separate for multiple.
- Sources Greenhouse/Ashby/Lever JDs with migrate/replace language.
- personas required when include\_personas=true.
- Billing mode: company (1 / company; \+1 per persona match when personas on -\> typically 2). Catalog credit\_cost\_per\_unit=1.
- volume=20-500; cadence=weekly.
