Find emails from up to 50 domains in one request.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/find-emails/bulk
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `urls` | Yes | Array of domains (max 50) |
| `limit` | No | Pages per domain (default 10) |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/find-emails/bulk \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"urls":["example.com","company.com"]}'
```

## Response

```json
{
  "results": [
    { "url": "example.com", "first_email": "contact@example.com", "status": "success" }
  ],
  "stats": {
    "total_urls": 2,
    "domains_with_emails": 1,
    "duration_ms": 8450
  }
}
```
