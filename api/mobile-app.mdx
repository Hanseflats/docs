Bulk check whether companies have a mobile app. Pass up to 200 domains per request.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/mobile-app
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domains` | Yes | Array of company domains (max 200). Aliases: `urls`, `company_domains`. A single `domain` / `company_domain` / `company_website` string is also accepted. |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/mobile-app \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domains":["hubspot.com","notion.so"]}'
```

## Response

```json
{
  "success": true,
  "results": [
    {
      "domain": "hubspot.com",
      "resolved_domain": "hubspot.com",
      "has_mobile_app": "Yes",
      "company_name_rb": "HubSpot"

    {
      "domain": "notion.so",
      "resolved_domain": "notion.so",

```
