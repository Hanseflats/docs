Pull a Substack publication archive (posts \+ author bylines) as structured JSON.

## Endpoint

```text
POST https://app.lens.ly/api/v1/scrapers/substack
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `subdomain` | Conditional | Publication subdomain (e.g. `lenny`) |
| `url` | Conditional | Publication URL (`{sub}.substack.com` or custom domain) |
| `page` / `max_pages` | No | Archive pagination |
| `limit` / `per_page` | No | Posts per page (default `25`) |
| `find_linkedin` / `find_email` | No | Enrich authors (`find_email` uses Substack profile → social LinkedIn → RB waterfall) |
| `webhook_url` / `delivery` | No | Async delivery options |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/scrapers/substack \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"subdomain":"lenny","max_pages":1}'
```
