Missing/thin team plus open roles in a hiring department (RevenueBase).

## Endpoint

```
POST https://app.lens.ly/api/v1/company/team-hiring
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |
| `lacking` | Yes | Department that may be missing/thin |
| `hiring` | Yes | Department to check for open roles |
| `lacking_mode` | No | zero (default) or thin |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/team-hiring \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"acme.com","lacking":"marketing","hiring":"sales"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "acme.com",
    "lacking_department": "marketing",
    "hiring_department": "sales",
    "lacks_team": true,
    "is_hiring": true,
    "open_roles": 3
  },
  "credits_used": 1
}
```
