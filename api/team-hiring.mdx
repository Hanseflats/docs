Bulk check missing/thin team plus open roles in a hiring department. Shared params apply to all domains.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/team-hiring
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domains` | Yes | Array of company domains (max 200) |
| `lacking` | Yes | Department that may be missing/thin |
| `hiring` | Yes | Department to check for open roles |
| `lacking_mode` | No | `zero` (default) or `thin` |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/team-hiring \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domains":["acme.com","example.io"],"lacking":"marketing","hiring":"sales","lacking_mode":"zero"}'
```

## Response

```json
{
  "success": true,
  "results": [
    {
      "domain": "acme.com",
      "lacking_department": "marketing",
      "hiring_department": "sales",
      "lacks_team": true,
      "is_hiring": true,
      "role_count": 0,
      "open_roles": 3
    }
  ],
  "stats": {
    "total_domains": 2,
    "matched": 1,
    "credits_used": 2
  }
}
```
