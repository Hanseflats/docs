Launch via the [Audiences API](/api/audiences) with `play_slug: "missing-team-hiring-companies"`.

Find US companies missing one department (e.g. no marketers) that are actively hiring another (e.g. sales / AEs / demand gen). Uses role counts \+ open-role counts — one query, no job scrape. Optionally match buyer personas.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `lacking` | Yes | marketing \| sales \| customer\_success \| security \| devops \| engineering \| it. Example: `marketing` |
| `hiring` | Yes | sales \| account\_executive \| marketing \| demand\_generation \| customer\_success \| security \| devops \| it \| business\_development. Example: `sales` |
| `lacking_mode` | No | zero (default) or thin (≤1). Example: `zero` |
| `companySizes` | No | Comma-separated size ids, e.g. 11-50,51-200. Example: `11-50,51-200` |
| `include_personas` | No | true to match contacts after companies. Default false.. Example: `false` |
| `personas` | No | Job titles when include\_personas is true. Example: `Founder, VP Sales` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `missing-team-hiring-companies` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/missing-team-hiring-companies \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: missing-team-hiring-companies-1" \
  -d '{
  "play_slug": "missing-team-hiring-companies",
  "source_params": {
    "lacking": "marketing",
    "hiring": "sales"
  },
  "number_of_leads": 100,
  "title": "Companies Without a Team That Are Hiring"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Companies Without a Team That Are Hiring",
    "play_slug": "missing-team-hiring-companies",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=companies`.

## Credits

**1 credit per company** found; **2** if a buyer persona is matched. Charged as results are found, not at launch.

## Notes

- Requires lacking \+ hiring departments. hiring: sales|account\_executive|marketing|demand\_generation|customer\_success|security|devops|it|business\_development.
- Defaults: lacking=marketing, hiring=sales, lacking\_mode=zero.
- Billing mode: company (1 / company; \+1 per persona match when personas on -\> typically 2). Catalog credit\_cost\_per\_unit=1.
- volume=20-2000; cadence=monthly.
