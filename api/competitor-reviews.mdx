Launch via the [Audiences API](/api/audiences) with `play_slug: "competitor-reviews"`.

Paste company websites and/or direct G2 / TrustRadius / Trustpilot review URLs (comma-separated). Websites are auto-matched to review pages; direct URLs are used as-is. Collects recent reviews (past 8 months), optional star filter, and finds reviewers on LinkedIn.

## Endpoint

```text
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `websites` | Yes | Company websites and/or direct G2 / TrustRadius / Trustpilot URLs, comma-separated.. Example: `hubspot.com, https://www.g2.com/products/salesforce/reviews` |
| `find_work_email` | No | When enabled, look up each reviewer work email and employer when possible.. Example: `true` |
| `max_rating` | No | Only include reviews at or below this rating. Leave blank for all.. Example: `3` |
| `max_leads` | No | Stop after this many resolved leads across all platforms.. Example: `200` |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `competitor-reviews` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/competitor-reviews \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: competitor-reviews-1" \
  -d '{
  "play_slug": "competitor-reviews",
  "source_params": {
    "websites": "hubspot.com, https://www.g2.com/products/salesforce/reviews"
  },
  "number_of_leads": 100,
  "title": "Competitor Reviews (G2 + TrustRadius + Trustpilot)"
}'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Competitor Reviews (G2 + TrustRadius + Trustpilot)",
    "play_slug": "competitor-reviews",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=leads`.
