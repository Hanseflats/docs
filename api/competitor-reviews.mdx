---
title: "Competitor reviews"
description: "Audience play that scrapes G2, TrustRadius, and Trustpilot reviews and resolves reviewers to LinkedIn profiles."
---

Launch via the [Audiences API](/api/audiences) with `play_slug: "competitor-reviews"`.

Paste company websites and/or direct G2 / TrustRadius / Trustpilot review URLs. Websites are auto-matched to review pages; direct URLs are used as-is. Collects recent reviews (past 8 months), optionally filters by star rating, and finds reviewers on LinkedIn.

## Endpoint

```
POST https://app.lens.ly/api/v1/audiences
```

## Source params

| Parameter | Required | Description |
| --- | --- | --- |
| `websites` | Yes | Company websites and/or direct G2 / TrustRadius / Trustpilot URLs, comma-separated |
| `find_work_email` | No | When true, look up each reviewer's work email and employer when possible |
| `max_rating` | No | Only include reviews at or below this star rating (1–5) |
| `max_leads` | No | Stop after this many resolved leads across all platforms |

Also pass top-level audience fields:

| Parameter | Required | Description |
| --- | --- | --- |
| `play_slug` | Yes | Must be `competitor-reviews` |
| `number_of_leads` | No | Target volume (default `100`) |
| `title` | No | Display name |
| `cadence` | No | `daily`, `weekly`, or `monthly` for recurring runs |
| `delivery_webhook_url` | No | Webhook for delivery |

Inspect the live schema anytime:

```bash
curl https://app.lens.ly/api/v1/audiences/plays/competitor-reviews \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/audiences \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: competitor-reviews-1" \
  -d '{
    "play_slug": "competitor-reviews",
    "source_params": {
      "websites": "hubspot.com, https://www.g2.com/products/salesforce/reviews",
      "max_rating": "3",
      "find_work_email": "true",
      "max_leads": "200"
    },
    "number_of_leads": 100,
    "title": "Competitor reviewers"
  }'
```

## Response

```json
{
  "audience": {
    "id": "a1b2c3d4-...",
    "title": "Competitor reviewers",
    "play_slug": "competitor-reviews",
    "apify_status": "pending"
  },
  "status": "triggered"
}
```

Poll with `GET /api/v1/audiences/{id}`. Export with `GET /api/v1/audiences/{id}/export?type=leads`.
