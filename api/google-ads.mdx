Check Ads Transparency for active Google ads and sample creatives.

## Endpoint

```
POST https://app.lens.ly/api/v1/company/google-ads
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/google-ads \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"notion.so"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "notion.so",
    "running_google_ads": true,
    "advertiser_name": "Notion Labs",
    "creative_count": 12
  },
  "credits_used": 1
}
```
