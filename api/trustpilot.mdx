Scrape Trustpilot into structured JSON. Two modes:

- **Reviews** (default) — review cards for a business page
- **Company** — business profile (`mode=company` or `find_persona`) for decision-maker lookup

## Endpoint

```text
POST https://app.lens.ly/api/v1/scrapers/trustpilot
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `url` | Yes | Company website **or** Trustpilot URL |
| `mode` | No | `reviews` (default) or `company` |
| `page` | No | Start page for reviews (default `1`) |
| `max_pages` | No | Review pages to scrape (default `1`). `>1` runs async |
| `find_linkedin` | No | Reviews mode only — resolve reviewer LinkedIn (\+1 per success) |
| `find_persona` | No | Company mode — find personas at the business domain |
| `persona_titles` | With `find_persona` | Job titles (max 3) |
| `find_email` | — | **Not supported** on Trustpilot (reviews lack employer identity) |
| `webhook_url` | No | Delivery webhook for async jobs |
| `delivery` | No | `chunks` (default), `item`, or `complete_only` |
| `webhook_batch_size` | No | Default `50`, max `100` |

## Modes

### Reviews (default)

Returns reviewer names and review text. Optional `find_linkedin`. `find_email` is rejected (`enrich_not_supported`).

### Company (`mode=company` or `find_persona`)

Returns a company-shaped row with `domain`. Use `find_persona` \+ `persona_titles` to pull decision-makers.

## Example — company \+ persona

```bash
curl -X POST https://app.lens.ly/api/v1/scrapers/trustpilot \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "hubspot.com",
    "mode": "company",
    "find_persona": true,
    "persona_titles": ["CEO", "VP Marketing"]
  }'
```

## Example — reviews

```bash
curl -X POST https://app.lens.ly/api/v1/scrapers/trustpilot \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "hubspot.com",
    "max_pages": 3,
    "find_linkedin": true,
    "webhook_url": "https://hooks.example.com/scraper",
    "delivery": "chunks"
  }'
```

## Credits

**1 credit per page** (reviews) or **1 credit** for company profile. `find_linkedin` / `find_persona` add \+1 each when found.

Charged against your prepaid Billing balance at **3¢ per credit**.

## Errors

| Status | Meaning |
| --- | --- |
| `400` | Invalid fields or unsupported enrich flag (`enrich_not_supported`) |
| `401` | Invalid API key |
| `402` | Insufficient billing balance |
| `404` | Company / page not found |

Response shapes (sync, async, poll, results, webhooks): [Scrapers overview](/api/scrapers).
