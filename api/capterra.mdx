title: "Capterra"

---

Look up a Capterra product page as structured JSON. Pass a website (we resolve the product) or a Capterra URL.

## Endpoint

```text
POST https://app.lens.ly/api/v1/scrapers/capterra
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `url` | Conditional | Company website **or** `capterra.com/p/{id}/{slug}` |
| `product_id` \+ `slug` | Conditional | Explicit product id and slug |
| `find_linkedin` / `find_email` | No | Enrich reviewers when present |
| `webhook_url` / `delivery` | No | Async delivery options |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/scrapers/capterra \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://www.capterra.com/p/135002/HubSpot-CRM/"}'
```
