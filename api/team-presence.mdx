Check whether a department team is missing or thin (RevenueBase).

## Endpoint

```
POST https://app.lens.ly/api/v1/company/team-presence
```

## Request parameters

| Parameter | Required | Description |
| --- | --- | --- |
| `domain` | Yes | Company domain |
| `lacking` | Yes | Department: marketing, sales, engineering, … |
| `lacking_mode` | No | zero (default) or thin |

## Example request

```bash
curl -X POST https://app.lens.ly/api/v1/company/team-presence \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"acme.com","lacking":"marketing","lacking_mode":"zero"}'
```

## Response

```json
{
  "success": true,
  "data": {
    "resolved_domain": "acme.com",
    "lacking_department": "marketing",
    "lacks_team": true,
    "role_count": 0
  },
  "credits_used": 1
}
```
