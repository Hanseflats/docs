Pipeline (Lens) provides prepaid APIs for company enrichment and audience building.

## Base URL

```
https://app.lens.ly
```

## Authentication

All authenticated requests require:

```
Authorization: Bearer YOUR_API_KEY
```

Or:

```
x-api-key: YOUR_API_KEY
```

## Quick example

```bash
curl -X POST https://app.lens.ly/api/v1/company/mobile-app \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain":"stripe.com"}'
```

## For AI agents

Start at [`/llms.txt`](https://app.lens.ly/llms.txt) or [`GET /api/v1/meta`](/agents/meta), then register → verify → billing setup. See [Agent self-serve](/agents/overview).
