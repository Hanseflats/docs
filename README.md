# Pipeline API — Mintlify docs

Upload this folder to Mintlify (or push to a GitHub repo connected to Mintlify).

## Preview locally

```bash
npm i -g mint
cd mintlify-docs
mint dev
```

## Contents

- `docs.json` — site config + navigation
- `*.mdx` — pages (Getting started, Agents, Reviews, Enrichments, Guides)

Base URL: https://app.lens.ly
